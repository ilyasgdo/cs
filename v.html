Setup Unity & Meta Quest

Guide de configuration rapide pour macOS

1. Préparation du Matériel & Compte

  • Mode Développeur : Créer une organisation sur dashboard.oculus.com.
  • Application Mobile : Dans l'app Meta Horizon (Smartphone), aller dans Paramètres du casque > Mode Développeur et l'activer.
  • Logiciel INDISPENSABLE sur Mac : Installer le Meta Quest Developer Hub (MQDH) pour assurer la connexion USB.

2. Configuration Unity Hub

Dans l'onglet Installs, ajouter les modules suivants à votre version d'Unity :

  • Android Build Support
  • Android SDK & NDK Tools
  • OpenJDK

3. Création du Projet

  • Utiliser le template Universal 3D (URP).
  • File > Build Settings : Sélectionner Android et cliquer sur Switch Platform.
  • Edit > Project Settings > XR Plug-in Management :
    • Cliquer sur Install XR Plug-in Management.
    • Dans l'onglet Android (petit robot), cocher Oculus.

4. Configuration de la Scène (VR Toolkit)

  • Window > Package Manager : Installer XR Interaction Toolkit (choisir Unity Registry).
  • Note : Accepter le redémarrage d'Unity pour le nouveau "Input System".
  • Hierarchy : Supprimer la Main Camera.
  • Hierarchy : Clic droit > XR > XR Origin (VR).

5. Build and Run

  1. Brancher le casque au Mac via USB-C.
  2. Vérifier dans le casque l'autorisation de Débogage USB.
  3. Dans Unity (Build Settings), cliquer sur Refresh à côté de Run Device.
  4. Sélectionner le casque et cliquer sur Build and Run.
Astuce Mac : Si le casque n'est pas détecté, ouvrez le Meta Quest Developer Hub. Une fois que le point est vert dans MQDH, il apparaîtra dans Unity.
